# Food Nutrition API

## Installation

Requires Node and NPM on host machine.

Go to [nodejs.org](https://nodejs.org/en/).

### Install npm dependencies

From project directory

```
$ npm i
```

## Run Server

From project directory

```
$ npm start
```

You should now be able to view app at http://localhost:3000/

## Run Test

From project directory

```
$ npm test
```
